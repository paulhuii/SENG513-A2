Battleship Game

Target Platform

Desktop

Game Genre

Strategy, Board Game

Game Objective

Sink all of your opponent's ships before they sink yours.

Rules

  1. Each player has a grid to place their ships. Top Grid first, then bottom grid (follow alerts)
  2. Players take turns guessing coordinates to attack. (player will choose opponent grid first, then opponent can choose on players grid)
  3. A hit or miss is marked on the grid. (Blue = miss, Red = Hit)
  4. The game continues until one player's fleet is entirely sunk. (5 hits and game is over!)
